lingualeo-translator
====================================

Translates and adds words with Lingualeo service.
Requires python 2.7+, 3.4+


Dependencies
------------------------------------

* PyYAML
* requests
* colorama

Install
------------------------------------

    git clone https://github.com/trezorg/lingualeo-translator.git
    cd lingualeo-translator
    python setup.py install

    or

    pip install . --user


Using
------------------------------------

    lingualeo.py --help
